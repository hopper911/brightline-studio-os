# Product Requirements

High-level product definition. See [vision.md](../00-overview/vision.md) for strategic context and the full department model.

## Summary

Bright Line Studio OS is an AI-powered operating system for a photography studio. It provides room-based workspaces, each with a dedicated AI agent, connected by shared services (events, drafts, approvals, projects).

## Core value

- Centralize studio operations in one app
- AI assistants that draft and classify but do not act without approval
- Safe, local-first architecture
