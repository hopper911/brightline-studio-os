# Bright Line Studio OS Architecture

## Goal
A local-first, Clawbot-like but safe AI operating system for a photography studio.

## Core concepts
- room-based UI
- sandboxed agents
- tool registry
- event logs
- approvals
- sessions
- local SQLite
- read-only image analysis in v1

## Current phase
Phase 1 foundation:
- route scaffold
- studio map UI
- SQLite setup
